# 🎰 Lucky Draw 樂透抽號機

> 從 1~50 號中抽出 3 個幸運號碼，附彩球動畫、抽號紀錄與統計。

## 🎮 Live Demo

👉 [立即抽號](https://lucky-draw-three.vercel.app)

## ✨ 功能特色

- 🎱 **彩球動畫**：旋轉 → 逐一揭曉，附光暈與彈跳效果
- 🎊 **彩帶特效**：抽號完成後撒彩帶慶祝
- 📋 **抽號紀錄**：自動儲存最近 30 筆（使用 localStorage）
- 📊 **統計面板**：總抽次、最熱號碼、最冷號碼
- 🎨 **5 色彩球**：紅、藍、金、綠、紫，每球獨立配色
- 📱 **響應式**：手機平板皆適用

## 🕹️ 使用方式

點擊「✨ 開始抽號」，系統自動從 1~50 不重複抽出 3 個號碼，附動畫揭曉。
